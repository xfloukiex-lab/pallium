The problem

The fediverse gave people sovereign social media and self-hosted storage. But the newest, most invasive layer of the internet — AI — still lives almost entirely inside a handful of Big-Tech clouds. Every prompt, document, and memory you give an AI assistant leaves your control.

Pallium closes that gap. It's a single, easy-to-deploy package that runs a complete private AI on a machine you own — a home server or a cheap VPS.

Local model inference, an agent memory you keep, private search over your own files, a hardened gateway that lets the AI use tools safely, and a sandboxed agent that can't phone home unless you let it. Personal freedom and portability at the core — your AI is yours to move, no lock-in.

The stack

Seven composable pieces, deployed declaratively (NixOS). Bring it up with one command; move it between hosts when you want.

Run

Local model serving

Your own LLM inference on localhost. Nothing leaves the box. Runs on modest hardware.

Remember

Agent memory & knowledge

A memory and knowledge store you own — with a portable format so you can move your "AI self."

Recall

Private retrieval

Search across your own files and notes locally. No cloud, no third-party index.

Protect

Secure tool-gateway

Every tool the AI calls is checked: anti tool-poisoning, injection scanning, secret redaction, and your approval for anything dangerous.

Contain

Sandbox & egress control

The agent runs in an OS-enforced sandbox with a network kill-switch. A misbehaving agent can't reach out or damage the host.

Control

Admin console

Configure, watch, and approve from one place — and export your whole stack to another machine.

What's genuinely new

Self-hosting individual AI parts exists. Running a capable, tool-using agent safely and portably on your own box, as a turnkey stack, does not. That's the research.

One policy, three boundaries

A single declarative rule that spans the tool, OS, and network layers at once — one place to say what the agent may touch.

Reproducible private-AI packaging

Models + memory + tools + security shipped as one reproducible NixOS module set.

Portable AI state

Export and move your memory, knowledge, and config between hosts. Anti-lock-in for the AI layer.

Safe by default

The defaults that make it safe for a non-expert to run an autonomous agent at home.

Open source

Everything Pallium ships is under a recognised open-source license (Apache-2.0). Security tools earn trust by being auditable — so the core is free and open, forever.

  1. Free & open core. The full stack and the secure gateway are open source. Run it, read it, change it.
  2. Built to a mission. Service portability and personal freedom at the core — a sovereign alternative to Big-Tech AI clouds.
  3. Made by people who ship. VektorGeist builds local-first, privacy-preserving, open-source tools — released to real registries, hardened, and maintained.
